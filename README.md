# 光遇 · 拼音星河

一个结合《光遇》艺术风格的沉浸式拼音学习网站。

## 功能特点

- 🎨 **光遇风格设计**：柔美渐变色、星光背景、光影效果
- 🔊 **完整音频功能**：TTS发音、录音跟读、音量控制
- 🎮 **拼音游戏**：互动问答、星星奖励系统
- 📱 **响应式设计**：适配手机、平板、电脑
- ✨ **视觉效果**：动画过渡、粒子特效、光影交互

## 访问地址

https://haiwuyutu.github.io/Serena/

## 技术栈

- HTML5 / CSS3 (CSS变量、渐变、动画)
- JavaScript (ES6+)
- Web Audio API
- Speech Synthesis API
- MediaDevices API (录音)
